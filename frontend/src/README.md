# Frontend Source Structure

This directory contains the Streamlit frontend source code.

## Structure

- `pages/` - Streamlit page modules for different tabs
- `components/` - Reusable UI components
- `utils/` - Utility functions for API communication and helpers

## Files to be created

- `app.py` - Main Streamlit application entry point
- `pages/filter_processing.py` - Image filter processing tab
- `pages/detection.py` - Disease detection tab
- `components/image_uploader.py` - File upload widget
- `components/filter_selector.py` - Filter selection UI
- `components/result_display.py` - Image result display
- `components/health_card.py` - Health information cards
- `utils/api_client.py` - Backend API client
- `utils/ui_helpers.py` - Vietnamese message formatting
