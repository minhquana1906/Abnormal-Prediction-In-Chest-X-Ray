# YOLO Model Weights Directory

This directory stores the trained YOLOv11s model weights for chest X-ray abnormality detection.

## Files

- `yolov11s_finetuned.pt` - Fine-tuned model weights (created after training in notebooks/)

## Notes

- This is the ONLY persistent storage directory in the project
- All other image processing happens in memory
- Model file size: ~20-30MB
